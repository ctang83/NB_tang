  • WORD中的通配符主要有： *代表0至任意个字符； ^?代表任意字符； ^#代表任意数字； ^$代表任意字母； ^&代表查找内容（在替换框使用，相当于原有内容基础上附加内容）。

  • excel 批量插入空行 的方法：

  1. 用alt+F11进入VBE编程
  2. 双击左边你的工作表名称
  3. 复制以下代码： Sub 添加4空行() Do While ActiveCell <> “” ActiveCell.Offset(1, 0).Resize(4, 1).EntireRow.Insert shift:=xldowm ActiveCell.Offset(5, 0).Select Loop End Sub
  4. Alt+Q返回Excle
  5. 将光标停在需要插行的下一行按Alt+F8插入，遇到空单元格会停止插入。
  6. 注：这个操作无法撤消。
  7. 若想加更多行修改(4,1)和(5,0)数据
